## 异常数据处理
### 3$\sigma$法则
> 剔除利润率偏差极大的数值
68% 的数据点位于均值 μ 和均值 ±1标准差 σ之间。
95% 的数据点位于均值 ±2标准差σ之间。
99.7% 的数据点位于均值 ±3标准差σ之间。
在实际应用中，这个规则用于识别异常值或偏离正常范围的数据点。如果某个数据点超出了均值 ±3标准差的范围，通常被认为是异常值或极端值。
